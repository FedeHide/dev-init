# Dev-Init

## Building

```bash
cd dev-init && ./dev-init.sh <output>
```

If you encounter issues running this script, try granting it execution permissions:

```bash
chmod +x dev-init.sh
```

## Features

- ✅ automates the setup process for my personal development projects
